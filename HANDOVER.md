## How to Run

See [README.md](README.md) for detailed instructions. Quick summary:

**Docker (recommended):**
```bash
export GEMINI_API_KEY=your-key
docker-compose up --build
```

**Manual:**
```bash
# Terminal 1
cd backend && python -m venv venv && source venv/bin/activate && pip install -r requirements.txt && python run.py

# Terminal 2
cd frontend && npm install && npm run dev
```

## Zip Manifest — What to Keep vs Delete

### DELETE before zipping (saves ~500MB+):

| Path | Reason | Approx Size |
|------|--------|-------------|
| `frontend/node_modules/` | Rebuilt via `npm install` | ~300MB |
| `backend/venv/` | Rebuilt via `pip install -r requirements.txt` | ~200MB+ |
| `backend/__pycache__/` (all) | Auto-generated bytecode | ~1MB |
| `backend/app/routers/__pycache__/` | Auto-generated | ~1MB |
| `backend/app/services/__pycache__/` | Auto-generated | ~1MB |
| `backend/.pytest_cache/` | Test cache | ~1MB |
| `backend/test_db.sqlite` | Dev-mode database (recreated at runtime) | ~1MB |
| `backend/interview.db.json` | Mock DB fallback (recreated at runtime) | ~1KB |

**Quick cleanup command (run from project root):**

```powershell
# Windows PowerShell
Remove-Item -Recurse -Force frontend\node_modules -ErrorAction SilentlyContinue
Remove-Item -Recurse -Force backend\venv -ErrorAction SilentlyContinue
Get-ChildItem -Recurse -Directory -Filter __pycache__ | Remove-Item -Recurse -Force
Remove-Item -Recurse -Force backend\.pytest_cache -ErrorAction SilentlyContinue
Remove-Item backend\test_db.sqlite -ErrorAction SilentlyContinue
Remove-Item backend\interview.db.json -ErrorAction SilentlyContinue
```

```bash
# macOS/Linux
rm -rf frontend/node_modules backend/venv
find . -type d -name __pycache__ -exec rm -rf {} + 2>/dev/null
rm -rf backend/.pytest_cache backend/test_db.sqlite backend/interview.db.json
```

## Environment Variables (.env)

Create `backend/.env` with:

```env
USE_MONGODB=false
MONGODB_URL=mongodb+srv://user:pass@cluster.mongodb.net/interview_db
GEMINI_API_KEY=your-api-key-here
GEMINI_MODEL=gemini-2.5-flash
JWT_SECRET_KEY=change-me-in-production
AI_SAFE_MODE=false
INFERENCE_DEVICE=cuda
```

Set `USE_MONGODB=false` for instant demo without MongoDB setup.
