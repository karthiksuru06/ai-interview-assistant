## 1. Abstract
The **Smart AI Interview Assistant** is a real-time coaching system designed to democratize professional interview preparation. By leveraging a **Hybrid Fusion Architecture** of Computer Vision (CNN) and Audio Analysis (LSTM), the system provides instant, multimodal feedback on a candidate's non-verbal communication. This report validates that high-fidelity AI coaching is viable on consumer hardware, achieving **<20ms latency** on an NVIDIA RTX 4060.

## 2. Methodology: Hybrid Fusion Architecture

To replicate the intuition of a human interviewer, the system processes two concurrent data streams:

1.  **Visual Stream (FER)**:
    *   **Model**: EfficientNet-B0 (Fine-tuned on FER2013).
    *   **Function**: Analyzes facial micro-expressions at 30 FPS.
    *   **Optimization**: Uses Mixed Precision (FP16) on Tensor Cores to minimize VRAM usage (~8ms inference).

2.  **Audio Stream (Paralinguistics)**:
    *   **Model**: Bidirectional LSTM.
    *   **Function**: Analyzes vocal prosody, pitch stability, and confidence vectors from MFCC features.
    *   **Latency**: Extremely fast inference (~6.5ms) due to optimized sequence length handling.

3.  **Sensor Fusion**:
    *   The backend fuses these probability distributions using a weighted average strategy, ensuring that a nervous facial expression is cross-referenced with vocal stability before flagging a "Low Confidence" alert.
